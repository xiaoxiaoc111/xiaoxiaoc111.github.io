0x00 Docker 概念

    • vm docker 虚拟机 容器 k8s hypervisor是什么？

      • 物理主机-> 虚拟机（vm） ->容器（docker）
        vm 和docker 的 优缺点?
      • vm 启动慢，可移植性差，专用系统
        k8s是什么?
      • k8s是容器编排（google开放）k8s 是docker之上的平台，采用docker实现底层容器相关操作
      • 内核命名空间
      • 控制组联合文件系统
      • kubernetes 是在docker之上的平台 -> the kubernets book
      • docker 引擎 ：企业版 和社区版 EE&CE
        • https://docs.docker.com/engine/release-notes/
        • https://github.com/moby/moby
      • 开放容器计划OCI
      • play with docker PWD
      • linux端 client-daemon交互是通过本地IPC/UNIX socket - /var/run/docker.sock

0x05 Docker Compose

    • docker compose 是用来管理多个docker容器来实现一个应用的管理
    • compose文件主要4个key ： version services networks volumes 其他
      • version ：并不是docker compose 和docker的版本 是compose文件的版本
      • services ： 定义不同的应用
      • networks : 指引docker 创建新的网络 默认是bridge drive属性自定义
      • volumes ： 指引docker 创建新的卷
