  • insatll sublime
    • 如何在Ubuntu 18.04中安装Sublime Text 3.2
#sublime
wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -
echo "deb https://download.sublimetext.com/ apt/stable/" | sudo tee /etc/apt/sources.list.d/sublime-text.list
sudo apt update && sudo apt install sublime-text

#要删除编辑器，请运行命令：
sudo apt remove --autoremove sublime-text
  • sublime 中文
    • 不用补丁，sublime text 3 本身 设置中文方法
 Tools->Install Package Control  - package control
 Tools->Command Palette...
 ipc ,点击 Install Package Control，调出 Package Control
 Preferences->Package Control 输入ip，点击“install Package” 输入clz
  • ubuntu登录页面用root账号
    • 关于 Ubuntu Linux 16.04中文版的 root 权限及桌面登录问题
    • Ubuntu启用root账户后，登录root账户时报错：读取/root/.profile时发现错误：mesg:ttyname failed:对设备不适当的ioctl操作 作为结果，会话不会被正确配置
cd usr/share/lightdm/lightdm.conf.d
sudo vim 50-unity-greeter.conf　
    greeter-show-manual-login=true       #手工输入登录系统的用户名和密码
    allow-guest=false                    #不允许guest登录

vi /root/.profile
mesg n”替换为“tty -s && mesg n
